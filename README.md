# Cohort Retention Dashboard (Tableau)

Interactive cohort analysis dashboard built in Tableau Public, featuring KPI tiles and a cohort retention heatmap for tracking user/customer retention over time.  

## Live Demo
- Tableau Public: https://public.tableau.com/views/Book1_17687536615850/Dashboard1?:language=en-US&publish=yes&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link

## What’s Included
- KPI: Total acquired customers (cohort size)
- KPI: Month‑1 retention
- KPI: Best Month‑1 cohort
- Retention heatmap (Cohort Month × Months Since)

## Build Notes (Implementation)
- Heatmap layout: Rows = Cohort Month, Columns = Months Since, Marks = Square, Color = Retention Rate.
- KPI tiles built as text-based sheets for clean dashboard layout.
